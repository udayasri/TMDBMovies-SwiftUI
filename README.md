# TMDBMovies-SwiftUI
TMDBMovies Sample implementation using Swift UI : MVVM 

## Features
- Show list of of genres.
- Show list of movies for a selected genre.
- Supports iOS / tvOS 
- Supports pagination
- Error handling

## Requirements

* `iOS` 16.0+
* `tvOS` 16.0+
* `Swift` 5.10
* `Xcode` 15.3

## Getting Started

- Add your `themoviedb.org` api key in TMDBConstants.
